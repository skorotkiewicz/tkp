# Variables & Constants

## Mutable Variables

```tkp
ijo name = 42
ijo message = "toki"
name = 100    // reassignment allowed
```

## Constants

```tkp
awen pi = 3.14
// pi = 3.0  → runtime error: cannot reassign constant
```

## Type Annotations

```tkp
ijo age: nanpa_kind = 25
ijo key: kipisi = 175.5
ijo name: sitelen = "name_A"
ijo active: lawa = kin
```

## Null Values

```tkp
ijo value = weka
toki(value)    // weka
```
