# Math Functions

| Function | Description | Example | Result |
|----------|-------------|---------|--------|
| `lili_nanpa(x)` | Square root | `lili_nanpa(16.0)` | `4.0` |
| `wawa_nanpa(x)` | Absolute value | `wawa_nanpa(-5)` | `5` |
| `suli_nanpa(base, exp)` | Power | `suli_nanpa(2, 10)` | `1024.0` |

All math functions accept both `nanpa_kind` and `kipisi` inputs.
